gpbayeskit

Gaussian process tools for Bayesian spatio-temporal modelling

Python ≥ 3.10 NumPy · SciPy · pandas MIT licence

gpbayeskit provides a composable toolkit for fitting, simulating, and visualising Gaussian process models over space and space-time. It implements the Matérn, confluent hypergeometric (CH), and Lagrangian covariance families, a self-describing parameter system that handles REML optimisation automatically, space-time data containers inspired by R’s spacetime package, and publication-ready diagnostic plots — all with a consistent API.

Installation

From source (recommended during development)

# clone and install in editable mode
git clone https://github.com/pulongma/gpbayeskit.git
cd gpbayeskit
pip install -e .

# with optional plotting dependencies
pip install -e ".[plot]"

# with development tools (pytest, ruff)
pip install -e ".[plot,dev]"

Dependencies

  • Required: numpy ≥ 1.24, scipy ≥ 1.10, pandas ≥ 1.5
  • Optional: matplotlib ≥ 3.7 for all plotting functions
Editable install note: If you see ImportError: cannot import name 'KernelBuilder' from 'gpbayeskit.kernels' after cloning, verify that every subdirectory under gpbayeskit/ contains an __init__.py. Without it Python treats the directory as a namespace package with no exports. Run pip install -e . --force-reinstall once all files are in place.

Quick start

Simulate data, fit a spatial GP, and evaluate predictions in a few lines:

import numpy as np
from gpbayeskit.simulation import simulate
from gpbayeskit import SpatialGP

rng  = np.random.default_rng(42)
X    = rng.uniform(0, 1, (100, 2))
sim  = simulate(X, kernel_type="matern52", phi=0.3, sigma2=2.0, seed=0)

X_tr, y_tr = X[:80], sim.y[:80]
X_te, y_te = X[80:], sim.y[80:]

gp = SpatialGP(X_tr, y_tr, kernel_type="matern52")
gp.fit()
gp.summary()

mean, var = gp.predict(X_te)
scores    = gp.score(X_te, y_te)
print(scores["rmse"], scores["coverage_95"])

Spatial GP

SpatialGP supports six kernel families and three anisotropy forms. Parameters are declared as class-level descriptors; fit() maximises the REML log marginal likelihood via L-BFGS-B. All fitting, prediction, scoring, and summary logic is inherited from BaseGP — subclasses only implement covariance().

Kernel families

matern
General Matérn
Smoothness ν is a free parameter; includes exp, 3/2, 5/2 as special cases
exp / matern32 / matern52
Fixed smoothness
ν auto-fixed to 0.5, 1.5, or 2.5 respectively
ch
Confluent hypergeometric
Adds tail-decay parameter α for heavy-tailed fields
gauss
Squared-exponential
ν = ∞; infinitely differentiable fields

Anisotropy forms

Formφ shapeDescription
isotropicscalarSingle range for all directions
ard(d,)Automatic relevance determination — one range per dimension
tensor(d,)Product kernel — separable across dimensions

Fitting with fixed and initial values

from gpbayeskit import SpatialGP

gp = SpatialGP(X_tr, y_tr, kernel_type="matern", kernel_form="ard")
gp.fit(
    fix_nu=1.5,        # fix smoothness; excluded from optimisation
    phi_init=0.3,      # starting value for the d-vector of ranges
    nugget_init=0.01,  # starting value for nugget
)

Fit keyword convention

ArgumentTypeEffect
fix_<name>floatFix to constant; excluded from optimisation
<name>_initfloat or arraySet starting value before optimisation

Spatio-temporal GP

SpatioTemporalGP extends BaseGP with Lagrangian and frozen-field covariance functions. The input array X has shape (n, d_space + 1) where the last column is time.

Models

ModelFormula C(h, u) / σ²
frozen_maternM( |h − uλ| / φ ; ν )
frozen_chCH( |h − uλ| ; ν, α, φ )
lagrangian_matern|I + ρ²Λ/2|−½ M( hu / φ ; ν )
lagrangian_ch|I + ρ²Λ/2|−½ CH( hu ; ν, α, φ )

The Lagrangian distance is hu = √((h − uλ)T(u²Λ + I)−1(h − uλ)), where λ = λ0[cos θ0, sin θ0] is the advection vector and Λ = U diag(λ1, λ2) UT is the deformation matrix.

Usage

from gpbayeskit import SpatioTemporalGP

# X: (n, 3) — columns s1, s2, t
gp = SpatioTemporalGP(X_tr, y_tr, model="lagrangian_matern", d_space=2)
gp.fit(
    fix_nu=1.5,
    fix_nugget=0.01,
    lam0_init=0.3,     # advection magnitude starting value
    theta0_init=0.0,   # advection direction (radians)
)
gp.summary()
mean, var = gp.predict(X_te)

Space-time data containers

gpbayeskit.stdata provides four container classes for real-world spatio-temporal data, inspired by R’s spacetime package (Pebesma 2012). All classes share a common base with to_long(), to_time_wide(), to_space_wide(), summary(), and cross-format conversion methods.

STF
Full grid
All n_loc × n_t combinations present. 3-D tensor (n_t, n_loc, n_var).
STS
Sparse grid
Same declared lattice as STF, but with missing entries tracked explicitly.
STI
Irregular
Fully arbitrary (s, t) pairs. Direct input to SpatioTemporalGP.
STT
Trajectories
Ordered paths partitioned by an ID column. Animal movement, floats, vehicles.

Three output formats

MethodRowsColumns
to_long()one per (s, t) observation[coord_names…, time, vars…]
to_time_wide()spatial locationstime points (one column per time step)
to_space_wide()time pointsspatial locations (one column per location)

STF → STI → model

from gpbayeskit.simulation import simulate_st
from gpbayeskit.stdata import STF, STI

# build STF directly from a simulation result
sim = simulate_st(n_space=20, t_values=np.linspace(0, 1, 6),
                   phi=0.25, lam0=0.4, seed=0)
stf = STF.from_simulation(sim)

# inspect
stf.summary()
stf.at_time(0.0)        # spatial snapshot as DataFrame
stf.at_location(0)     # time series at location 0
stf.to_time_wide()      # (n_loc, n_t) DataFrame — locs as rows
stf.to_space_wide()     # (n_t, n_loc) DataFrame — times as rows

# convert to STI for model fitting
sti = stf.to_sti()
train, test = sti.train_test_split(test_fraction=0.2, seed=1)
X_tr, y_tr  = train.to_model_arrays()   # (n, 3), (n, 1)

STT — trajectory data

from gpbayeskit.stdata import STI, STT

# each STI is one trajectory (already sorted by time)
trajs = [STI(coords_i, times_i, data=z_i, var_names=["z"])
         for coords_i, times_i, z_i in trajectory_data]

stt = STT.from_sti_list(trajs)
stt.summary()
stt.arc_lengths()              # total path length per trajectory
stt.bounding_boxes()           # spatial / temporal extents
traj0 = stt.get_trajectory(0)  # returns an STI

Simulation

Spatial simulation

from gpbayeskit.simulation import simulate

sim = simulate(
    X,
    kernel_type="matern52",
    phi=0.3, sigma2=2.0, nugget=1e-4,
    seed=42, return_latent=True,
)
sim.y       # (n, 1)  observed draw (with nugget noise)
sim.latent  # (n, 1)  noise-free latent draw
sim.cov     # (n, n)  full covariance matrix

Spatio-temporal simulation

simulate_st builds the full joint space-time covariance across all (space × time) locations and makes a single draw, so temporal correlations and the advection pattern are faithfully preserved.

from gpbayeskit.simulation import simulate_st

sim = simulate_st(
    n_space=20,
    t_values=np.linspace(0, 1, 6),
    model="lagrangian_matern",
    phi=0.25, nu=1.5, sigma2=1.0, nugget=1e-4,
    lam0=0.4, theta0=np.deg2rad(30),  # advect north-east at 30 deg
    rho=0.5, lambda1=1.5, lambda2=0.5,
    seed=42,
)
sim.field.shape    # (6, 20, 20) — one 2D snapshot per time step
sim.lam_vec        # advection vector lambda
PSD note: The simulation uses the u-dependent prefactor |A(u)|−½ where A(u) = (u/ρ)²Λ + I. This is the mathematically correct Lagrangian form (Stein 2005) that guarantees C(0, 0) = σ² and a positive-definite covariance matrix. A constant prefactor, as used in some parameterisations, is not generally positive semi-definite.

Plotting

All functions require matplotlib (install with pip install gpbayeskit[plot]) and return (fig, axes) without calling plt.show().

Log-likelihood profiles

from gpbayeskit.plotting import plot_loglik, plot_loglik_panel

# single profile: log-lik vs phi with nu and nugget fixed
grid, vals, ax = plot_loglik(
    X, y,
    param="phi", param_range=(0.05, 2.0),
    fixed_params={"nu": 1.5, "nugget": 0.01},
    kernel_type="matern", log_x=True,
)

# panel of profiles
specs = [
    {"param": "phi",    "param_range": (0.05, 2.0),
     "fixed_params": {"nu": 1.5, "nugget": 0.01}, "log_x": True},
    {"param": "nu",     "param_range": (0.5,  3.0),
     "fixed_params": {"phi": 0.3, "nugget": 0.01}},
    {"param": "nugget", "param_range": (1e-5, 0.1),
     "fixed_params": {"phi": 0.3,  "nu": 1.5},   "log_x": True},
]
fig, axes = plot_loglik_panel(X, y, specs, kernel_type="matern")

Space-time covariance surfaces

from gpbayeskit.plotting import plot_cov_st_1d, plot_cov_st_2d

params = dict(phi=0.5, nu=1.5, lam0=0.4, theta0=0.2,
              rho=0.8, lambda1=1.2, lambda2=0.4, theta_Lambda=0.3)

# 1D contour: C(h, u) for a 1D spatial domain
# advection appears as a tilted ridge
fig, ax = plot_cov_st_1d(params,
                          h_range=(-2, 2), u_range=(0, 2),
                          model="lagrangian_matern",
                          title="Lagrangian covariance C(h, u)")

# 2D panel: C(h1, h2; u) at several temporal lags
# white arrow marks the advection displacement u*lambda
fig, axes = plot_cov_st_2d(params,
                            h_range=(-2, 2),
                            u_values=[0, 0.5, 1.0, 2.0],
                            model="lagrangian_matern",
                            title="Spatial covariance slices")

# both functions also accept a fitted SpatioTemporalGP directly
fig, axes = plot_cov_st_2d(fitted_gp,
                            h_range=(-2, 2),
                            u_values=[0, 1, 2])

Spatio-temporal realisations

from gpbayeskit.simulation import simulate_st
from gpbayeskit.plotting   import plot_st_realizations

sim = simulate_st(n_space=25, t_values=np.linspace(0, 1, 6),
                   lam0=0.4, theta0=np.deg2rad(30), seed=0)

fig, axes = plot_st_realizations(
    sim, ncols=3,
    show_tracer=True,      # gold dot drifts with advection velocity
    show_velocity=True,    # arrow shows advection direction
    title="Lagrangian GP realisation — NE advection",
)
